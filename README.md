# User Management System 🔐

## 🔧 Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Angular/React (TBD)
- JWT Authentication

## 🚀 Features
- Register / Login (JWT)
- Role-based Access (Admin/User)
- Admin can view & delete users
- User can update own profile

## 📂 Project Structure
- `Backend/` - ASP.NET Core Web API
- `Frontend/` - Angular/React UI
- `SQL/` - SQL scripts
